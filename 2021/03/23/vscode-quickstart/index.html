简单易懂的vscode入门指南

Posted on 2021-03-23

Visual Studio Code是由微软带头开发的开源编辑器，有着丰富的功能，出色的性能与强大的可扩展性，因而受到许多人的欢迎。
本文会对vscode做一个简要的介绍，并收集一些受欢迎的vscode教程供大家参考。
希望阅读本文的你能喜欢上vscode。

1. vscode可以用来做什么

  • 写代码。写一切你能想到和想不到的代码。
    • 借助丰富的扩展插件，vscode支持众多主流/小众的编程语言；
    • 调试运行，语法高亮，自动补全，单元测试，商业IDE支持的功能它全都有，并且不用费心思搞破解，全是免费的；
  • 写论文写文档
    • 借助扩展，vscode同样支持了latex与markdown文档的编写，并均有着不俗的体验。
      • latex: 使用vscode编写LaTex
      • markdown: 搭建 Markdown 舒适写作环境（VSCode，含扩展盘点）
  • 代码仓库管理
    • vscode原生支持git，可直接在编辑器内部对仓库进行管理，无需命令行或额外的git工具（如smart git）
    • 借助gitlens, git graph等扩展，还可使得通过git协作更加方便。
      • Git魔术——VSCode中如何使用Git（以github作为远端存储库）
  • 玩
    • 装上这几个 VSCode 插件后，上班划水摸鱼不是梦
    • 太赞了，VSCode 上也能画流程图了！
    • 又一款骚气满满的 VSCode 插件，只为吹爆开发者彩虹屁而生！
    • 如何将宇宙最强vscode打造为刷题神器

2. 为什么选择vscode

  • 比起商业IDE（如pycharm，eclipse，visual studio等），vscode的优点有：
    • 轻量化：启动速度更快，性能更好
    • 一体化：在同一个地方编辑你的代码与文档，进行调试，提交，协作；
    • 强大的编辑功能：多行同时编辑、快速选择文本、搜索编辑器、regex搜索/替换等；
      • https://code.visualstudio.com/docs/editor/codebasics
    • UI：简洁美观的界面，并且支持自定义主题、图标、字体、颜色等等；
      • vscode主题推荐
    • 支持打开jupyter notebook文件
    • 支持远程开发
      • VSCode Remote 体验 | 远程Linux环境开发真香
      • 使用vscode进行远程炼丹
    • 支持Docker
      • Docker+VSCode配置属于自己的炼丹炉
  • 为什么不选vscode
    • 对于一个（或几个）特定的语言，IDE往往有着更好的支持，并能提供更多功能；
    • 如果你IDE对于你来说更加顺手，当然也可选择不用vscode；
      • 但我仍推荐你在编辑少数文件，或一些IDE不支持的语言时使用vscode，这可以为你节省不少时间。

3. vscode的下载、安装与中文配置

可以在官网下载到最新版vscode：

  • 官网下载：https://code.visualstudio.com/
    • 有Windows，Mac，Linux三个平台的版本可选，按照自己的需求下载即可。
  • 安装：
    • 总之就是点下一步和完成啦
  • 配置中文：
    • 左边一排排里有一个扩展按钮，点进去
    • 在扩展面板搜索chinese
    • 安装跳出来的第一个，即Chinsese (Simplified) Language Pack
    • 重启vscode

4. vscode 数据科学（python）开发环境搭建

搜不到最新的教程，只好自己写一个了。

  • 首先，你需要自己安装python或者anaconda。

    • 如果pip默认源的下载速度太慢，可以考虑更换为清华大学镜像站tuna

    临时使用

    设为默认

  • 然后，在vscode的扩展面板中搜索并安装如下几个扩展：

    • Python Extension Pack
      • 包含5个python开发相关的扩展
    • Pylance
      • 自动导入依赖，智能提示，自动补全，类型推断等
    • Jupyter
      • 支持notebook
      • （可能已经随着python扩展一起安装）

  • 最后，使用pip或者conda安装jupyter

    • pip install jupyter
    • 或 conda install jupyter
